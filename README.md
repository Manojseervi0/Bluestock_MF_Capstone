# Mutual Fund Analytics Platform

## Overview
This project is an end-to-end Mutual Fund Analytics Platform developed as part of the Bluestock Fintech Capstone Project.

The platform collects mutual fund data from AMFI and MFAPI, processes it through an ETL pipeline, stores it in SQLite, computes advanced analytics, and visualizes insights through Power BI dashboards.

## Features
- Automated Data Ingestion
- Data Cleaning & Preprocessing
- SQLite Data Warehouse
- Exploratory Data Analysis
- Performance Analytics (CAGR, Sharpe Ratio, Sortino Ratio)
- Risk Analytics (VaR, CVaR, Maximum Drawdown)
- Investor Analytics
- Recommendation Engine
- Interactive Power BI Dashboards

## Technologies Used
- Python
- Pandas
- NumPy
- SQLite
- Power BI
- Git & GitHub
